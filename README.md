# iPFS 

This repository contains the code, benchmarks and data for the article "A software-based focus system for wide-field optical microscopy" also called iPFS (imPerfect Focus System).
The article is available from arXiv, https://arxiv.org/abs/2511.20890

The structure of this repo as follows:
- [Code](./Code/): The main iPFS script, a prototype working with 3 point interpolation, plus the script used to benchmark the system.
- [Figures](./Figures/): The figures used for the making the panels, 
- [Notebook](./Notebook): To reproduce the figures from the article using preprocessed data please head to this section for more information.

Data availability: 
- Preprocessed data (reduced size) is available as a github release.
- Raw data (image time-lapses, raw metadata, etc.) is available upon reasonable request to the authors.
