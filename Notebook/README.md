The layout of this folder is as follows:


- [figures_paper.ipynb](figures_paper.ipynb): allows to generate most figures which are contained in the paper. Some familiarity with running jupyter notebooks is expected. 
-[analysis_functions.py](analysis_functions.py): functions used by the notebook
